# DearMenu 需求文档

## 项目信息
- **项目名称**: DearMenu（亲爱的菜单）
- **创建日期**: 2026-04-03
- **负责人**: 老公（开发者）
- **用户**: 老婆大人

## 背景故事
老公想为老婆开发一个日常点菜小程序，让老婆每天不用纠结吃什么，一键随机推荐，还能收藏自己喜欢的菜，记录历史点餐。

## 需求概述

### 目标用户
- 主要用户：老婆大人
- 使用场景：每天日常点餐选择
- 使用设备：手机（移动端网页）
- 使用频率：每天多次

### 核心价值
1. **减少选择困难** - 随机推荐功能帮助快速决定
2. **记录喜好** - 收藏功能保存老婆喜欢的菜
3. **历史追溯** - 记录每次点的什么
4. **多设备同步** - 数据云端化，换设备不丢失

## 功能需求

### 1. 用户系统

#### 1.1 注册
- 用户名（唯一标识）
- 密码（加密存储）
- 无需邮箱验证（简化流程）

#### 1.2 登录
- 用户名 + 密码登录
- JWT Token 认证
- Token 持久化（LocalStorage）

### 2. 分类管理

#### 2.1 预设分类
初始提供以下分类：
- 零食（🍪）
- 饮料（🥤）
- 奶茶（🧋）
- 水果（🍎）
- 热菜（🍳）
- 熟食（🍖）

#### 2.2 分类操作
- 添加自定义分类（名称 + 图标）
- 编辑分类
- 删除分类（需确认，防止误删）
- 排序（拖拽或序号）

### 3. 菜品管理

#### 3.1 菜品属性
| 属性 | 类型 | 说明 |
|------|------|------|
| 名称 | 文本 | 必填，最长50字 |
| 分类 | 外键 | 必填，关联分类表 |
| 标签 | 多选文本 | 可选，如：辣、清淡、油炸等 |
| 评分 | 1-5整数 | 必填，5星制 |
| 创建时间 | 时间戳 | 自动生成 |

#### 3.2 标签预设
辣、清淡、油炸、甜、咸、清蒸、红烧、生吃、加热即食

#### 3.3 菜品操作
- 添加菜品
- 编辑菜品
- 删除菜品
- 按分类筛选
- 按名称搜索

### 4. 收藏功能

#### 4.1 收藏操作
- 在菜品列表点击收藏按钮
- 已收藏显示实心图标
- 再次点击取消收藏

#### 4.2 收藏列表
- 展示所有收藏的菜品
- 支持取消收藏
- 支持按名称搜索

### 5. 历史记录

#### 5.1 记录内容
- 菜品名称
- 点菜时间
- 分类信息

#### 5.2 历史展示
- 按时间倒序排列
- 显示日期分组
- 支持清空历史（需确认）

### 6. 随机推荐

#### 6.1 随机逻辑
- 从所有菜品中随机
- 可选择指定分类
- 避免连续重复推荐

#### 6.2 随机结果
- 展示随机选中的菜品
- 显示菜品详情
- 一键收藏
- 确认点这个（记录到历史）

## UI/UX 需求

### 设计风格
- **整体风格**: 高端精致
- **视觉感受**: 简洁、大方、优雅
- **目标用户感受**: 被宠爱的感觉

### 配色方案
```
主色调: #FF6B6B (珊瑚红 - 温暖有爱)
辅助色: #4ECDC4 (薄荷绿 - 清新)
强调色: #FFE66D (金色 - 精致)
背景色: #FAFAFA (浅灰白)
文字色: #2C3E50 (深灰)
```

### 字体
- 主字体: 系统默认（移动端优化）
- 标题: 加粗，字号 18-24px
- 正文: 常规，字号 14-16px

### 布局
- 底部 Tab 导航（5个标签）
  - 首页
  - 分类
  - 菜品
  - 收藏
  - 我的
- 卡片式列表
- 圆角按钮
- 适度留白

### 动效
- 页面切换：滑动过渡
- 按钮点击：轻微缩放
- 收藏：心跳动画
- 随机选择：滚动转盘效果

## 技术需求

### 前端
- React 18
- Vant Mobile UI
- Vite 构建
- Zustand 状态管理
- React Router 路由

### 后端
- Python FastAPI
- PostgreSQL 15
- Docker 部署
- JWT 认证

### 数据库
- Docker 容器运行
- 用户隔离（RLS 或应用层过滤）
- 自动备份（待定）

## 非功能需求

### 性能
- 首屏加载 < 2秒
- 操作响应 < 500ms

### 安全
- 密码加密存储（bcrypt）
- JWT 过期机制
- SQL 注入防护

### 兼容性
- iOS Safari 最新版
- Android Chrome 最新版
- 微信内置浏览器

## 优先级

| 优先级 | 功能 |
|--------|------|
| P0 | 登录/注册、菜品 CRUD、随机推荐 |
| P1 | 分类管理、收藏功能、历史记录 |
| P2 | UI 优化、动画效果 |
| P3 | 数据备份、导出 |

4. **正式上线** - 部署发布
