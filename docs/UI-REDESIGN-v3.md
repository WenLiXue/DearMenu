# DearMenu v3.0 UI 重设计文档

## 设计定位

**从「可爱工具」升级为「精致、克制、有情绪价值的情侣产品」**

---

## 一、设计哲学

### 核心理念
- **克制**：Less is More，每屏不超过 2 个主色
- **温柔**：柔和的色彩、有呼吸感的间距
- **轻高级**：iOS 原生质感、Notion 留白、小红书卡片结构
- **仪式感**：情侣互动、情绪价值

### 设计参考
- iOS 原生应用（克制 + 系统感）
- Notion（留白 + 秩序）
- 小红书（卡片 + 轻量）
- Airbnb（卡片 + 情绪）

## 二、色彩系统 v3.0

### 2.1 主色调（降低饱和度）

| 角色 | 色值 | 用途 |
|------|------|------|
| 老婆主色 | `#F06565` | 主按钮、强调元素（之前 #FF6B6B） |
| 老婆渐变 | `linear-gradient(135deg, #F06565 0%, #E8A0A0 100%)` | 仅用于关键主按钮 |
| 老公主色 | `#3CBAB2` | 主按钮、强调元素（之前 #4ECDC4） |
| 老公渐变 | `linear-gradient(135deg, #3CBAB2 0%, #8DD4D0 100%)` | 仅用于关键主按钮 |

### 2.2 背景体系（iOS 风）

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#F5F5F7` | iOS 风格中性灰 |
| 卡片背景 | `#FFFFFF` | 纯白卡片 |
| 分割线 | `#F0F0F2` | 极淡分隔 |
| 输入框背景 | `#F5F5F7` | 融入背景 |

### 2.3 强调色策略

| 用途 | 色值 | 使用频率 |
|------|------|----------|
| 随机推荐 | `#FFF8E7`（暖白）+ `#F5D76E`（强调） | 随机页面专用 |
| 成功色 | `#3CBAB2` | 完成状态 |
| 警告色 | `#F5A623` | 中等提醒 |
| 危险色 | `#E85A5A`（降低饱和） | 删除/取消，慎用 |
| 收藏色 | `#F06565` | 收藏动效专用 |

### 2.4 文字色阶

| 层级 | 色值 | 用途 |
|------|------|------|
| 主文字 | `#1D1D1F` | 标题、重要信息 |

| 次要文字 | `#86868B` | 辅助信息、时间戳 |
| 占位文字 | `#AEAEB2` | 输入框占位符 |

## 三、字体系统 v3.0

### 3.1 字号层级（精简至 5 种）

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 大标题 | 28px | 700 | 页面主标题 |
| 标题 | 17px | 600 | 卡片标题、模块标题 |
| 正文 | 15px | 400 | 主要内容 |
| 辅助 | 13px | 400 | 辅助说明、时间 |
| 标签 | 12px | 500 | 标签、小徽章 |

### 3.2 文字规范
- **标题**：不使用渐变色，改为深色 + 字重
- **emoji**：降级为辅助元素，不作为视觉焦点
- **英文**：使用系统默认字体栈

## 九、动效系统 v3.0

### 9.1 统一规范
- **时长**：200ms ~ 300ms
- **缓动**：`cubic-bezier(0.25, 0.46, 0.45, 0.94)`（ease-out）
- **点击反馈**：`scale(0.96)`

### 9.2 关键动效

| 动效 | 实现 |
|------|------|
| 卡片悬停 | `transform: translateY(-2px); box-shadow 增强` |
| 按钮点击 | `transform: scale(0.96)` |
| 页面切换 | `opacity: 0→1, translateX(20px→0)` |
| 收藏心跳 | `scale: 1→1.2→1, 300ms` |
| 随机翻牌 | `rotateY: 0→180deg, 400ms` |
| 完成打勾 | `scale: 0→1.2→1, 200ms` |

## 十三、目标达成

重设计后的 DearMenu：

| 感受 | 实现方式 |
|------|----------|
| 干净 | 大量留白、克制用色 |
| 放松 | 柔和色彩、低饱和 |
| 喜欢 | 情绪文案、仪式感 |
| 温柔 | 圆角、阴影、动效 |

**用户打开后的感受**：这不是一个工具，而是一个有温度的家庭助手。
